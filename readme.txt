1. Download the WebCrawler.jar file.
2. In the terminal, go to the directory in which WebCrawler.jar is stored
and execute the following command: 
"java -jar WebCrawler.jar https://www.bbc.co.uk/"
where "https://www.bbc.co.uk/" can be any other valid url that you want to start
searching from.
3. Make sure that the url you supply is valid (i.e. protocol and 
domain are present and correct) otherwise the program will not accept it.
4. You should see the desired output printed in the terminal.
